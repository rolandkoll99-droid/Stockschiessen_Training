📖 Bedienungsanleitung

Stockschützen Trainingssystem

1. Erster Zugriff & Login

  1. Öffnen Sie die Datei in einem Browser
  2. Geben Sie das Standard-Passwort ein: admin123
  3. Klicken Sie auf "🔓 Zugang" oder drücken Sie Enter

2. Das Hauptfenster

  • Kopfzeile: Titel, Spieleranzahl, Einstellungen, Reset
  • Punktestand: Gesamtpunkte + gewonnene Spiele
  • Kehren-Bereich: Alle Kehren mit Punkte-Buttons (0–6)
  • Fußzeile: Spiel werten, Neustart, Kehre löschen, Logout, Hilfe

3. So wird trainiert

  • Spieleranzahl: Über das Dropdown "👥" einstellen (3–6)
  • Kehrenanzahl: Über "Kehren" Dropdown wählen (4/6/8)
  • Siege bis Trainingssieg: Über "Siege" Dropdown (2–6)
  • Punkte eintragen: Klicken Sie auf einen Zahlen-Button (0–6) für das jeweilige Team
  • Automatische Wertung: Wenn alle Kehren ausgefüllt sind, wird das Spiel automatisch gewertet
💡 Tipp: Die aktuelle Kehre wird durch einen grünen Rahmen markiert. Abgeschlossene Kehren haben einen orangen Rahmen.

4. Sonderfunktionen

FunktionButtonWirkung
Kehre löschen"Kehre Löschen"Löscht eine einzelne Kehre
Spiel zurücksetzen"Neues Spiel Starten"Löscht alle Kehren, Siege bleiben
Kompletter Reset↺ (roter Button)Setzt alles zurück (Punkte + Siege)
Logout"🚪 Logout"Sperrt die Anwendung

5. Teamnamen ändern

  • Geben Sie in den Textfeldern neben "⚪ Team 1" und "🔴 Team 2" die gewünschten Namen ein
  • Die Änderung wird sofort übernommen und gespeichert

6. Admin-Bereich – Passwort ändern

  1. Klicken Sie auf das Zahnrad-Symbol (⚙️) unten rechts
  2. Geben Sie aktuelles Passwort ein (Standard: admin123)
  3. Geben Sie neues Passwort ein (mindestens 4 Zeichen)
  4. Bestätigen Sie das neue Passwort
  5. Klicken Sie auf "🔑 Passwort ändern"
⚠️ Achtung: Bei Passwortverlust gibt es keine Wiederherstellung!

7. Tastenkürzel

TasteFunktion

8. Technische Hinweise

  • Browser: Alle aktuellen Browser (Chrome, Firefox, Edge, Safari)
  • Geräte: Desktop, Tablet, Smartphone (responsive)
  • Datenspeicherung: Lokal im Browser – keine externen Server
  • Datei: Einzelne HTML-Datei – keine Installation nötig
